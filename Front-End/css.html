CSS (Cascading Style Sheets)

CSS is used to style and layout HTML elements on a webpage. It controls the colors, fonts, spacing, alignment, and overall visual appearance of web content.

Key Features:

  • Separates content (HTML) from design (CSS).
  • Allows responsive design with media queries.
  • Supports animations and transitions.
  • Can apply styles globally or to specific elements.

CSS Syntax:


selector {
    property: value;
}

Example:


p {
    color: blue;
    font-size: 16px;
}

Types of CSS:

  • Inline CSS: inside HTML tags (not recommended for large projects)
  • Internal CSS: within a <style> tag in the HTML head
  • External CSS: in separate `.css` files linked via <link> tag

Example HTML + CSS:


<!DOCTYPE html>
<html>
<head>
    <style>
        body {
            background-color: lightgray;
        }
        h1 {
            color: green;
        }
    </style>
</head>
<body>
    <h1>Hello CSS!</h1>
</body>
</html>
